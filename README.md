<h1>AVIASYS</h1>
<p>AVIASYS — это веб-приложение для управления сотрудниками, цехами и задачами. Оно позволяет создавать, редактировать и отслеживать задачи в разных цехах предприятия. Система поддерживает роли администратора и сотрудников с авторизацией по логину и паролю.</p>
<h2>Инструкция по просмотру БД.</h2>
<p>Моя база данных работает через SQLite, и ssms не поддерживает его, чтобы просматривать и редактировать базу данных, нужно использовать DB Browser (https://sqlitebrowser.org/dl/). Когда вы установили программу, откройте DB Browser(SQLite) и вверху найдите Open DataBase. Нажмите на него, и в файле программы найдите файл aviasys.db (это и есть бд), и откройте его с помощью программы, и всё вы можете просматривать и редактировать бд.</p>
